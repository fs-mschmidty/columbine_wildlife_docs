# Milkweed Field Maps

The milkweed inventory map intended to house all unofficial and official milkweed surveys on the San Juan National Forest. The purpose of the map is to document where there is milkweed on the SJNF, including which species, density, and habitat. Please include images at all survey locations, both official and unofficial. 

## Map Layers and Description

### Milkweed Inventory Line

This map layer is intended for unofficial milkweed surveys. The observer will record a line that tracks them as they move along a survey area.

**Fields:**

1. Observer Name (required)

2. Date and Time (required)

3. Is Milkweed Present? (Required)

    - This is the most important field in the survey. If you have recorded a long line and a majority of the line does not have milkweed, please start a new line where the milkweed is. If you are unsure, we would rather you record unknown/other than no, so a wildlife biologist can later verify. 
    - Options:
        - Yes
        - No
        - Unknown/Other
